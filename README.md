# CareerVision AI - Career Prediction Portal

## Overview

CareerVision AI is a full-stack web application that helps students predict suitable career paths using Machine Learning. Users can create an account, log in securely, upload resumes, view prediction history, and receive AI-based career recommendations.


## Features

* User Signup & Login
* Secure Password Hashing (Flask-Bcrypt)
* Session Management
* AI Career Prediction
* Resume Upload
* Prediction History
* User Profile
* MySQL Database Integration
